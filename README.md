<h2>Overview</h2>
Wasil is a platform designed to help users exchange knowledge and real experiences about courses in various fields ‚Äî such as UI/UX, programming, and more.

<h2>Project Phases</h2>

### Phase 1: Wireframing & Prototyping 
- Low-fidelity wireframes (Web + Mobile)  
- Interactive prototypes  
- Iterations based on user feedback  

### Phase 2: UI Design 
- Design system: colors, typography, iconography, components  
- High-fidelity responsive screens  
- Accessibility: contrast ratios, readable typography, Arabic + English support  

### Phase 3: Testing & Validation 
- Usability testing  
- Feedback analysis  
- Compliance with government standards, security, performance  

### Phase 4: Final Delivery 
- High-fidelity prototypes (Web + Mobile)  
- Documentation: Design System, User Flows, Style Guide  
- Final presentation with outcomes & recommendations
